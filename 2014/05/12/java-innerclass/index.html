简介

简单地说，内部类是放在一个类内部的类。内部类核心提要如下：

  • 内部类可以很好地实现隐藏。一般的非内部类，不允许使用private、protected修饰，但内部类可以。
  • 内部类可以访问外部类的所有方法与属性。静态内部类只能访问外部类的静态属性与方法。
  • 内部类仍然是一个独立的类。编译之后会内部类会被编译成独立的.class文件，但是前面冠以外部类的类命和$符号。

静态内部类

静态内部类中”静”的含义是该内部类可以像其他静态成员一样，在没有外部类对象时也能够访问它。静态嵌套类不能访问外部类的非静态成员和方法。
